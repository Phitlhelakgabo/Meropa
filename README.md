##  Project Overview
This project is a multi-page website developed for Meropa Casino and Entertainment World. The purpose of the website is to provide information about casino gaming, entertainment, and services while offering a user-friendly experience.


##  Objectives
- Promote Meropa Casino and its services
- Provide information about gaming options
- Create an easy-to-navigate website
- Apply responsive design principles
- Improve user experience using CSS

## Website Structure
The website consists of the following pages:

- index.html → Home page
- about.html → Information about the casino
- gaming.html → Casino games and entertainment
- contact.html → Contact details and form
- style.css → External stylesheet for design


##  Technologies Used
- HTML (Structure of the website)
- CSS (Styling and layout)


##  Features Implemented
- Multi-page website navigation
- Responsive design using CSS
- Styled navigation menu
- Responsive images
- Contact form
- Footer on all pages
- Consistent layout and design


##  How to Run the Project
1. Download or copy all files into one folder
2. Open the folder
3. Double-click on index.html
4. Use navigation links to browse the website


##  User Experience (UX)
The website was designed with the user in mind:
- Simple and clean layout
- Easy navigation across all pages
- Readable text and clear headings
- Accessible images with alt text
